Syracuse (Collatz) Sequence

Easy loops,if-else,patterns

Problem Description

The Syracuse (or Collatz) sequence is defined as follows: given an initial integer greater than 0, repeatedly apply the following operations until the integer becomes 1:

  • If the number is even, divide it by 2.
  • If the number is odd, multiply it by 3 and add 1.

Your program must display the Syracuse sequence of the given number N and stop when the value 1 is reached.

Examples

Example:

For Example:
Input: 5

- Explaination:
Start with 5
5 is odd → 5 * 3 + 1 = 16
16 is even → 16 / 2 = 8
8 is even → 8 / 2 = 4
4 is even → 4 / 2 = 2
2 is even → 2 / 2 = 1
- Sequence stops at 1

Output: 5 16 8 4 2 1

Hints

Use a while loop

Apply if-else conditions

Ensure input is greater than zero
