Loop increment with step size (i += n)

Fundamentals Beginner

Overview

This is commonly used when we need to process every Nth character instead of every character.

Detailed Explanation

In programming, loops usually move one step at a time using i++

using i += n increases the loop variable by n on each iteration

this helps us directly access every Nth element without unnecessary checks

How it works ->

the loop starts from index 0

on each iteration, the index jumps by n

the loop stops when the index exceeds the string length

When to use ->

selecting every Nth character from a string

skipping elements in arrays

optimizing loops by avoiding unnecessary iterations

Key benefit ->

cleaner code

better performance

Code Examples

for (int i = 0; i < str.length(); i += n) {


    // process every Nth character


}
