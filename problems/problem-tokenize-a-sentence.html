Tokenize a Sentence

Easy java, strings, stringtokenizer

Problem Description

In Java the StringTokenizer class is used to break a string into tokens based on specified delimiters

You are given a sentence as a string. Your task is to ->

Split the sentence into individual words using StringTokenizer

Print each word on a new line in the order they appear.

Examples

Example:

Input->
Java is fun to learn

Output ->
Java
is
fun
to
learn
