Bank Account Balance Manager

Medium oop, classes, methods, encapsulation

Problem Description

Design a Java class named BankAccount that represents a simple bank account.

The class should:

Store the account holder name

Store the account number

Store the current balance

Implement methods to:

Deposit a given amount

Withdraw a given amount

Retrieve the current balance

Withdrawal should only be allowed if the balance is sufficient.

Examples

Example 1:

Input:

Initial Balance = 1000
deposit(500)
withdraw(300)


Output:

Balance after deposit: 1500
Balance after withdrawal: 1200


Explanation:
Deposit adds to balance, withdrawal subtracts if sufficient funds exist.

Example 2:

Input:

Initial Balance = 500
withdraw(800)


Output:

Withdrawal failed
Balance remains: 500


Explanation:
Withdrawal not allowed due to insufficient balance.
